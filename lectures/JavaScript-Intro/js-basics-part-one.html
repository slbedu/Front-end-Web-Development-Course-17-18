Примитивни типове

  • булеви
  • null
  • undefined

Пример

let num = 5,
    emptyObj = null,
    notDefined,
    myBool = true,
    myText = "Hello World!",

console.log(num); // 5
console.log(emptyObj); // null
console.log(notDefined); // undefined
console.log(myBool); // true
console.log(myText); // Hello World!
console.log(sym); // Symbol(foo)

Референтни типове - обекти

  • Неподреден списък от свойства, всяко от които има име и стойност
  • В JavaScript почти всичко е обекти (освен примитивните типове)
  • Присвояването на стойност отговаря на записването на указател към истинския обект в паметта

objectA["foo"] = "bar";

Оператори

Аритметични оператори

  • + - събиране
  • - - изваждане
  • * - умножение
  • / - делене
  • % - целочислено делене
  • ++ и -- - increment and decrement

Примери

const a = 5;
const b = 4;

console.log(a + b); // 9
console.log(a - b); // 1
console.log(a * b); // 20
console.log(a / b); // 1.25
console.log(a % b); // 1

console.log(a++); // ?
console.log(b--); // ?

Оператори за присвояване

  • =
  • +=
  • -=
  • *=
  • /=
  • %=

Примери

let result = 0;

console.log(result += 5); // 5
console.log(result -= 3); // 2
console.log(result *= 6); // 12
console.log(result /= 3); // 4
console.log(result %= 2); // 0

Логически оператори

  • && - логическо И
  • || - логическо ИЛИ
  • ! - NOT

Примери

const a = true;
const b = false;

console.log(a && b); // false
console.log(a || b); // true
console.log(!a); // false

Short circuit evaluation

Оператори за сравнение

  • ==
  • !==
  • ===
  • !==
  • <
  • >
  • <=
  • >=

Примери

const a = 5;
const b = 4;

console.log(a == b); // false
console.log(a !== b); // false
console.log(a > b); // true
console.log(a < b); // false

A little QUIZ :)

Побитови оператори

  • & - Bitwise AND
  • | - Bitwise OR
  • ^ - Bitwise XOR
  • ~ = Bitwise NOT
  • << - Bitwise Left shift
  • >> - Bitwise Right shift

Примери

const a = 3;
const b = 10;

console.log(a & b); // 2
console.log(a | b); // 11
console.log(a ^ b); // 9
console.log(~a); // -4
console.log(b >> 1); // 5
console.log(a << 1); // 6

Условни оператори

  • if
  • if-else
  • else if
  • switch-case

Примери

const a = 3;
const b = 10;

if (a === 3) {
    console.log(3);
}

if (b === 3) {
    console.log(3);
} else {
    console.log(10);
}

Още примери

const a = 3;
const b = 10;

if (a === 10) {
    console.log(3);
} else if (a === 5) {
    console.log(5);
} else if (b === 10) {
    console.log(10);
}

Още примери

const a = 3;
const b = 10;

switch (a) {
    case 3:
        console.log(3);
        break;
    case 5:
        console.log(3);
        break;
    default:
        console.log(":)");
}
